BrainBursa



Description

This course is a great introduction to both fundamental programming concepts and the Python
programming language. By the end, you'll be familiar with Python syntax and you'll be able to
put into practice what you'll have learned in a final project you'll develop locally.

Program

Python is a general-purpose, versatile and popular programming language. It's great as a first language
because it is concise and easy to read, and it is also a good language to have in any programmer's stack as
it can be used for everything from web development to software development and scientific applications.
